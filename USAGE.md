# DocVault 사용 가이드

처음 써도 막막하지 않게, 화면 순서대로 정리했습니다.

---

## 0. 가장 먼저: AI 키 등록 (1회)

변환은 AI가 하므로 **API 키가 필요**합니다.

1. 우측 상단 **⚙️ (설정)** 클릭 → **🔑 API 키**
2. 탭에서 프로바이더 선택 (Claude / GPT / Gemini / Custom)
3. 해당 사의 콘솔에서 발급한 **API 키**를 붙여넣고 **모델 선택** → 💾 저장
   - 비용을 아끼려면 기본값(Claude **Haiku 4.5**)을 그대로 두는 걸 추천합니다.
   - **BYOK(Bring Your Own Key) 전용**입니다. 키는 **이 브라우저에만 저장**되며, **서버는 키를 보관하지 않습니다** — 요청마다 브라우저가 보낸 키를 그대로 전달할 뿐입니다.

> 💡 종량제(API)만 지원합니다. claude.ai·ChatGPT **구독은 연결되지 않습니다.**
> 키가 없다면 각 사 개발자 콘솔(Anthropic / OpenAI / Google AI Studio)에서 발급하세요.

## 1. 문서 만들기 — 두 가지 방법

홈 화면 가운데의 탭에서 방식을 고릅니다.

### 방법 A. 📎 파일 업로드
1. 드롭존을 **클릭**하거나 파일을 **드래그**해서 올립니다. (여러 개 동시 선택 가능)
2. 아래 **변환 스타일**을 하나 고릅니다. **커스텀** 스타일을 고르면 프롬프트 입력창이 나타나 원하는 지시(예: "모든 소제목을 질문형으로 써줘")를 직접 적을 수 있습니다.
3. **⚡ 문서 변환** 클릭 → AI가 Markdown으로 정리.
   - 여러 파일이거나 큰 파일이면 변환 전에 **예상 청크 수·토큰·비용**을 보여주고 확인을 받습니다.
   - **긴 문서는 자동으로 청크 단위로 나눠 순차 변환**됩니다 — 아무리 길어도 잘리지 않고 끝까지 처리됩니다. 진행 중에는 "청크 2/5 변환 중..."처럼 진행 상황이 표시됩니다.
   - Claude를 사용 중이면 변환되는 내용이 **실시간으로 화면에 흘러나오는 미리보기**(접었다 펼 수 있음)를 볼 수 있습니다.
- 지원 형식: PDF, Word(DOCX), 스프레드시트(XLSX/XLS/ODS), TXT/MD/CSV/JSON, 코드 파일 등

### 방법 B. ✍️ 직접 작성  ← 파일 없이도 OK
1. **✍️ 직접 작성** 탭을 누릅니다.
2. 떠오르는 아이디어·메모·초안을 **자유 형식으로** 입력합니다.
3. 스타일을 고르고 **✨ 텍스트 변환** 클릭 → AI가 선택한 스타일의 문서로 정리.
   - 예: 두서없이 적은 회의 메모 → "보고서" 스타일로 변환하면 핵심요약·결정사항 형태로 정리됩니다.

> 변환이 끝나면 결과 문서가 **자동으로 열립니다.**

## 2. 변환 스타일 6종

| 스타일 | 언제 쓰나 |
|---|---|
| 💻 개발 문서 | 코드 블록·API 명세·기술 문서 |
| 📝 블로그 | 매력적인 제목 + 읽기 쉬운 서술형 |
| 📊 보고서 | 핵심요약(Executive Summary)·데이터 중심 |
| 📚 위키 | Obsidian용. YAML 머리말·`[[내부 링크]]`·태그 |
| 📖 공식 문서 | 엄격한 구조·정확한 정의 |
| ⚙️ 커스텀 | **직접 입력한 프롬프트**로 자유 변환 (예: 재변환 시에도 동일 프롬프트가 적용됩니다) |

---

## 3. 문서 열어서 할 수 있는 것 (상단 툴바)

문서를 열면 위쪽에 버튼들이 나타납니다.

- **👁 미리보기 / ✏️ 편집** — 보기/직접 수정 전환 (편집 후 💾 저장)
- **📄 원본** — 원본과 변환 결과를 **좌우로 나란히** 비교
- **🔄 재변환** — 같은 원본을 **다른 스타일**로 다시 생성 (기존 내용은 기록에 보존, 긴 문서도 동일하게 청킹 처리)
- **📋 복사** — 변환된 Markdown을 클립보드로
- **내보내기** — `.md` 파일로 다운로드
- **📋 로그** — 편집/재변환 이력
- **← 홈** — 문서를 닫고 홈 화면으로

---

## 4. 정리 & 찾기 (왼쪽 사이드바)

- **🔍 검색** + 아래 **필터 칩**(스타일별)과 **정렬**(최신·오래된·이름순)
- 하단에 저장 사용량(문서 수 · 용량 추정치) 표시
- 하단 **⬇️ Export**(전체 ZIP 백업) / **🗑**(전체 초기화 — 복구 불가, 주의)

---

## 5. 비용을 아끼는 팁 💰

변환은 토큰 종량제라 사용량만큼 과금됩니다.

- **기본 모델(Haiku 4.5)** 유지 — 변환 품질 대비 가장 저렴. 정말 어려운 문서만 Sonnet/Opus로.
- **일괄/대용량 변환 전 표시되는 예상 청크·비용**을 확인하고 진행하세요.
- 자동/백그라운드 호출은 없습니다 — **버튼을 누를 때만** 비용이 발생합니다.

## 자주 묻는 것

- **로컬에서 변환이 안 돼요** → `vite dev`에서는 AI 프록시(`/api/*`)가 동작하지 않습니다. 배포(Vercel) 환경에서 사용하세요.
- **구독으로 쓸 수 있나요?** → 아니요. API 키(종량제, BYOK)만 가능합니다.
- **데이터는 어디 저장되나요?** → 브라우저(IndexedDB)에만 저장됩니다. 여러 기기 동기화는 지원하지 않습니다 — 다른 기기로 옮기려면 ⬇️ Export(ZIP)로 백업 후 옮기세요.
- **API 키는 안전한가요?** → 서버는 키를 저장하지 않습니다(BYOK). 다만 브라우저에 저장된 값이라 공용 컴퓨터에서는 사용 후 초기화를 권장합니다.
