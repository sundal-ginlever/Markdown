📖 사용 가이드

    
처음 써도 막막하지 않게, 순서대로 정리했어요.

    

      
0. 가장 먼저 — AI 키 등록 (1회)

      
변환은 AI가 하므로 API 키가 필요합니다. 우측 상단 ⚙️ → 🔑 API 키에서 프로바이더(Claude/GPT/Gemini)를 고르고 키를 붙여넣은 뒤 저장하세요. 키는 이 브라우저에만 암호화 저장되며 서버로 전송되지 않습니다.

      
💡 종량제(API)만 지원합니다. claude.ai·ChatGPT 구독은 연결되지 않아요. 비용을 아끼려면 기본 모델(Claude Haiku 4.5)을 그대로 두는 걸 추천합니다.


      
1. 문서 만들기 — 두 가지 방법

      
📎 파일 업로드 — 드롭존을 클릭하거나 파일을 끌어다 놓고(여러 개 가능), 스타일을 고른 뒤 ⚡ 문서 변환. PDF·Word·엑셀·텍스트/코드 지원.

      
✍️ 직접 작성 — 파일이 없어도 됩니다. 떠오르는 아이디어·메모를 자유롭게 적고 스타일을 고른 뒤 ✨ 텍스트 변환하면 AI가 문서로 정리해 줍니다.


      
2. 변환 스타일 6종

      
    
        
  • 💻 개발 문서 — 코드·API·기술 문서
    • 
        
  • 📝 블로그 — 매력적 제목 + 읽기 쉬운 서술형
    • 
        
  • 📊 보고서 — 핵심요약·데이터 중심
    • 
        
  • 📚 위키 — Obsidian용(YAML·[[링크]]·태그)
    • 
        
  • 📖 공식 문서 — 엄격한 구조·정확한 정의
    • 
        
  • ⚙️ 커스텀 — 직접 정한 프롬프트로 자유 변환
    • 
      


      
3. 문서를 열면 (상단 툴바)

      
    
        
  • 👁 미리보기 / ✏️ 편집 — 보기·직접 수정
    • 
        
  • 📄 원본 — 원본과 변환 결과를 좌우 비교
    • 
        
  • 🔄 재변환 — 같은 원본을 다른 스타일로 다시
    • 
        
  • 📋 복사 / 내보내기 — 클립보드·.md 파일
    • 
        
  • 질의응답 — 이 문서에 대해 AI에게 질문
    • 
      


      
4. 정리 & 찾기 (왼쪽)

      
🔍 검색 + 필터 칩·정렬, 📁+ 폴더 생성, 하단 ⬇️ 전체 백업(ZIP) / 🗑 전체 초기화(주의).


      
5. AI 엔진 & 저장 위치

      
상단 AI 엔진 드롭다운에서 고른 엔진으로 변환됩니다(변환 완료 알림에 사용된 엔진 표시). 입력한 키가 있는 엔진을 선택하세요.

      
입력한 원본과 변환된 문서는 이 브라우저(IndexedDB)에 저장됩니다. 출력 언어는 상단 KO/EN 토글을 따릅니다.


      
6. 비용 아끼기 💰

      
    
        
  • 기본 모델 Haiku 4.5 유지 — 품질 대비 가장 저렴
    • 
        
  • 일괄 변환 전 예상 비용을 확인하고 진행
    • 
        
  • 같은 문서 반복 질문은 자동 캐싱으로 비용 절감(Claude)
    • 
        
  • 버튼을 누를 때만 비용 발생 — 자동 호출 없음
