## 1. 프로젝트 개요

DocVault는 **단일 HTML 파일로 동작하는 AI 문서 인텔리전스 도구**입니다. v2.0에서는 단순한 평면 목록을 넘어 **폴더 기반의 계층 구조**와 **대용량 안정성(IndexedDB)**을 확보했습니다.

### v2.0 핵심 설계 원칙
- **Scalability**: IndexedDB를 통한 수천 개의 문서 관리.
- **Organization**: 폴더와 정렬을 통한 지식 체계화.
- **Accessibility**: 장애가 없는 웹 환경 (Focus Trap, ARIA 지원).

## 3. 기술 아키텍처

### 대용량 영속성 (IndexedDB)
v2.0부터는 `localStorage`의 5MB 한계를 극복하기 위해 `IndexedDB`를 주 저장소로 사용합니다.
- `docs`: 변환된 Markdown 본문 및 메타데이터 저장.
- `raw`: 원본 파일 바이너리/텍스트 저장.
- `logs`: 문서별 수정 이력 저장.
- `settings`: 폴더 구조, 정렬 상태, AI 설정 보관.

### 접근성 (Accessibility)
- **Focus Trap**: 모달 오픈 시 포커스가 내부에서만 순환하도록 제어합니다.
- **Keyboard Friendly**: `Esc`로 모달 닫기, `Tab` 내비게이션 등 키보드 조작성을 최적화했습니다.

## 4. v2.0 신규 기능 — 폴더 및 정렬

### 폴더링 시스템 (Folder Management)
- **폴더 생성**: 사이드바 상단의 `📁+` 버튼으로 새 폴더를 생성합니다.
- **문서 이동**: 문서 항목의 📁 버튼을 클릭하여 원하는 폴더로 배정합니다.
- **계층 구조**: 폴더별로 문서를 그룹화하여 렌더링하며, 접기/펼치기 기능을 지원합니다.
- **미분류**: 폴더에 속하지 않은 문서는 자동으로 'Uncategorized' 섹션에 표시됩니다.

### 문서 정렬 (Document Sorting)
- **정렬 기준**: 검색창 옆 드롭다운을 통해 **최신순, 오래된순, 이름순**으로 목록을 정렬할 수 있습니다.
- **상태 유지**: 선택한 정렬 기준은 IndexedDB에 저장되어 다음 접속 시에도 유지됩니다.

## 11. 미래 로드맵 (Future Roadmap)

DocVault v2.0 이후의 발전 방향입니다.

- [ ] **드래그 앤 드롭**: 사이드바 내 문서와 폴더를 드래그하여 직관적으로 배치 이동.
- [ ] **AI 자동 분류**: 문서 내용을 분석하여 적절한 폴더를 AI가 추천하거나 자동 이동.
- [ ] **문서 간 링크 (Bi-directional)**: `[[문서명]]` 링크 시 양방향 연결 그래프 시각화.
- [ ] **다중 선택 모드**: 여러 문서를 선택하여 일괄 이동, 삭제, 내보내기 기능.
- [ ] **암호화 저장**: 민감한 문서 본문을 사용자 키로 AES-256 암호화하여 IDB에 저장.
